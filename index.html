CSCE 242: Web Applications

P. Elena Maddox

According to USC's Academic Bulletin Description of the course, CSCE 242 is a course that teaches "Web technologies to support client-server computing" and "Implementation of client-server applications."

The course description in Instructor Portia Plante's syllabus states that "Firstly, students will utilize HTML and CSS to design the front-end of a website; matching a given specification. Secondly, they will use JavaScript to add functionality to their website. Lastly, they will learn to utilize node and server side JavaScript to store and retrieve data from a server."
